104 class StaleElementReferenceException(WebDriverException):
105 """
106 Thrown when a reference to an element is now "stale".
107  
108 Stale means the element no longer appears on the DOM of the page.
109  
110  
111 Possible causes of StaleElementReferenceException include, but not limited to:
112 * You are no longer on the same page, or the page may have refreshed since the element
113 was located.
114 * The element may have been removed and re-added to the screen, since it was located.
115 Such as an element being relocated.
116 This can happen typically with a javascript framework when values are updated and the
117 node is rebuilt.
118 * Element may have been inside an iframe or another context which was refreshed.
119 """
120 pass
